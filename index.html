⚡ Connect Supabase

Supabase gives you a free PostgreSQL database. Your inventory is saved permanently and works on any device.

1

Create a free Supabase account

Go to supabase.com → Start for free → sign in → create a new project. Wait ~1 min for it to set up.

2

Create the tables

Click SQL Editor in the left sidebar, paste and run this:

-- Products table CREATE TABLE products ( id uuid DEFAULT gen_random_uuid() PRIMARY KEY, name text NOT NULL, qty integer DEFAULT 0, notes text, updated_at timestamptz DEFAULT now() ); -- Stock log table CREATE TABLE stock_log ( id uuid DEFAULT gen_random_uuid() PRIMARY KEY, product_id uuid REFERENCES products(id) ON DELETE CASCADE, product_name text, type text, qty integer, reason text, stock_after integer, created_at timestamptz DEFAULT now() ); -- Allow public access ALTER TABLE products ENABLE ROW LEVEL SECURITY; ALTER TABLE stock_log ENABLE ROW LEVEL SECURITY; CREATE POLICY "public_all" ON products FOR ALL USING (true) WITH CHECK (true); CREATE POLICY "public_all" ON stock_log FOR ALL USING (true) WITH CHECK (true);
3

Get your API keys

Go to Project Settings → API. Copy your Project URL and the anon / public key.
